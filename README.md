# primer-pr-practica

Repositorio de práctica para aprender a contribuir con Pull Requests.

## ¿Qué es un Pull Request?

Un Pull Request (PR) es una forma de proponer cambios a un repositorio. Permite que otros revisen el código antes de que se integre a la rama principal.

## Pasos para contribuir

1. Haz un fork del repositorio
2. Crea una rama nueva: `git checkout -b mi-mejora`
3. Realiza tus cambios y haz commit: `git commit -m "descripción del cambio"`
4. Sube la rama: `git push origin mi-mejora`
5. Abre un Pull Request desde GitHub
